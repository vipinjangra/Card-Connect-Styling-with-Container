# Card Connect - Flutter Assignment

This Flutter project demonstrates how to build a styled intro card using:

- Container
- Row
- Column
- Text
- CircleAvatar

## Features
- Rounded corners
- Box shadow
- Clean layout using Row and Column
- Profile avatar
